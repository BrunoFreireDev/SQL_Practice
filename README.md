# SQL Practice

Repositório dedicado aos meus estudos de SQL.

Aqui estão exercícios, consultas e práticas desenvolvidas durante meu aprendizado em banco de dados.

## Estrutura do repositório

- SELECTs → exercícios e consultas básicas
- JOINs → consultas envolvendo relacionamento entre tabelas
- AGREGAÇÃO → uso de funções como COUNT, SUM, AVG, GROUP BY e HAVING

## Tecnologias

- MySQL
- MySQL Workbench

## Objetivo

Praticar e evoluir no domínio de SQL através de exercícios e experimentação.
